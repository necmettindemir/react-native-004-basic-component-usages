
-- Base usage of components in React Native

This application shows some base approaches to component usages in React Native.


In this application Expo is used.
To run this app following commands must be run sequantially.


It is supposed that npm is intalled on your computer.

1) npm install -g expo-cli
2) npm install
3) npm start

4) Run any android simulator
   Android studio can be used.

5) Run any ios simulator.
   Xcode can be used.(on mac computer)
